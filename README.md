# Software Engineering

## 📌 Descripción del Proyecto

Este repositorio contiene el proyecto de **Ingeniería de Software**,
desarrollado como parte del trabajo académico por los alumnos:

-   Juan Manuel de Miguel
-   Yonathan Ismael Bautista
-   Hugo García

El objetivo del proyecto es el **desarrollo de una aplicación**,
aplicando los principios, metodologías y buenas prácticas de la
ingeniería de software, incluyendo análisis de requisitos, diseño,
implementación, pruebas y documentación.

------------------------------------------------------------------------

## 🎯 Objetivos

-   Aplicar conceptos fundamentales de ingeniería de software en un
    proyecto real.
-   Diseñar y desarrollar una aplicación funcional y escalable.
-   Trabajar de forma colaborativa utilizando control de versiones.
-   Documentar adecuadamente el proceso de desarrollo.
